## ðŸ”¹ Project Workflow

### 1. Data Loading (Python)
- Load CSV dataset using Pandas  
- Inspect structure and data types  

### 2. Exploratory Data Analysis (EDA)
- Summary statistics  
- Distribution of purchase amount and ratings  
- Category-wise and age-group-wise analysis  

### 3. Data Cleaning
- Handle missing values  
- Remove duplicates  
- Standardize column names and formats  

### 4. SQL Analysis
- Import cleaned data into database  
- Write queries for:
  - Total revenue  
  - Average purchase amount  
  - Revenue by category  
  - Customer count by age group  
  - Subscription analysis  

### 5. Power BI Dashboard
- Create KPIs:
  - Average Review Rating  
  - Average Purchase Amount  
  - Total Customers  

- Create Visuals:
  - Revenue by Category  
  - Sales by Category  
  - Purchase Amount by Age Group  
  - Customer Count by Age Group  
  - % of Customers by Subscription Status  

- Add Slicers:
  - Gender  
  - Subscription Status  
  - Shipping Type  
  - Category
